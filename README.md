# Your-Market

[Voir le site en ligne](https://your-market.netlify.app/)

## Description

Your-Market est un site e-commerce que j’ai développé et déployé pour explorer le développement web et le déploiement d’applications interactives.  
Le projet m’a permis de créer un site fonctionnel et responsive, tout en pratiquant la résolution de bugs et l’optimisation de l’expérience utilisateur.

## Fonctionnalités principales

- Catalogue de produits avec pages dynamiques
- Interface utilisateur interactive et responsive
- Gestion simplifiée du panier pour l’achat fictif
- Déploiement en ligne via Netlify

## Technologies utilisées

- HTML
- CSS
- JavaScript
- Netlify

## Compétences développées

- Conception et déploiement d’un site web complet
- Gestion d’interfaces utilisateur et navigation dynamique
- Analyse, débogage et optimisation du site
- Apprentissage rapide et adaptation à de nouvelles technologies

## Résultat

Un projet concret qui illustre ma capacité à apprendre rapidement et à appliquer de nouvelles compétences dans un environnement pratique.
